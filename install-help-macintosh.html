Installing & Running TrueGrain for Macintosh

Step 1. Download & install the application

If you havenâ€™t done so, already, click the link below to download a disk image containing the installer for TrueGrain.

TrueGrain-1.2.1.dmg (~5MB)

When the download is complete, find the disk image in your downloads folder (probably your Desktop), and mount the disk image by double-clicking on its icon. A new volume will appear in the Finder called TrueGrain containing a single installer program entitled TrueGrain.pkg. Double-click this installer program to begin the install process. Follow the instructions to complete installation. Along the way, it will ask you to authenticate. When installation is finished, you can eject the "TrueGrain" volume and discard the dmg file.

Step 2. Download & install film packs

After the TrueGrain application is installed, download and install each of these film packs in turn, in the exact same manner as above.

Film-Pack-0-for-TrueGrain-1.x.dmg (~213MB)

Film-Pack-1-For-TrueGrain-1.x.dmg (~184MB)

Film-Pack-2-for-TrueGrain-1.x.dmg (~223MB)

Film-Pack-3-for-TrueGrain-1.x.dmg (~206MB)

Film-Pack-4-for-TrueGrain-1.x.dmg (~242MB)

Step 3. Running TrueGrain

After you have installed the TrueGrain application and the film packs, you can run TrueGrain (look for it in your Applications folder or use Spotlight). The software will operate in demo mode until you purchase a registration key.

Obtaining TrueGrain on CD-ROM

TrueGrain is no longer available on CD-ROM. All distributables are served from Amazon S3 servers with superb presence world-wide. If your Internet connection is too slow to download the data, please arrange for a friend to do so for you.
