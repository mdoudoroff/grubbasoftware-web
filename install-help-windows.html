Installing & Running TrueGrain for Windows

Step 1. Download & install the application

If you havenâ€™t done so already, click the link below to download an installer for TrueGrain.

TrueGrain-v1.2.1.msi (~2MB - Feb/11/2009)

When the download is complete, locate the installer in your downloads directory (probably your desktop) and double-click it. Depending on your settings, Windows may produce a warning or two about installing software: choose the option to proceed with installation. When the install process begins, follow the instructions to completion. When installation is finished, you can discard the MSI file.

Step 2. Download & install film packs

After the TrueGrain application is installed, download and install each of these film packs in turn, in the exact same manner as above.

TrueGrain-Film-Pack-0-v1.0.msi (~214MB)

TrueGrain-Film-Pack-1-v1.0.msi (~185MB)

TrueGrain-Film-Pack-2-v1.0.msi (~224MB)

TrueGrain-Film-Pack-3-v1.0.msi (~207MB)

TrueGrain-Film-Pack-4-v1.0.msi (~244MB)

Step 3. Running TrueGrain

After you have installed the TrueGrain application and the film packs, you can run TrueGrain (look for it in your Start menu). The software will operate in demo mode until you purchase a registration key.

Obtaining TrueGrain on CD-ROM

TrueGrain is no longer available on CD-ROM. All distributables are served from Amazon S3 servers with superb presence world-wide. If your Internet connection is too slow to download the data, please arrange for a friend to do so for you.
